## 实验室研究方向
孔雨晴老师的实验室主攻 (TCS and Economics) 。简单来说，这里是为“没有标准答案的考试”制定评卷规则的地方。

以前大家做机器学习或者众包打分，都有个 Ground Truth，有了参考答案一切都好办。但在现实世界中，比如让大家给一篇文章评价好坏，或者让 LLM 充当裁判去评估其他模型的输出，往往没有绝对正确的答案。这个时候怎么去鼓励大家/模型说真话？这就需要用到机制设计（Mechanism Design）。这就是孔老师研究的“信息抽取与评估（Information Elicitation/Evaluation）”。

实验室会用到大量的信息论（Information Theory）、算法博弈论（Algorithmic Game Theory）和随机过程等纯理论工具。近年来，实验室也非常紧跟时代的步伐，探索了如何利用机制设计校准大语言模型（LLMs）的输出评判，以及在预测市场（Prediction Markets）里的神奇应用。
